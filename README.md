# kaizen (改善)
---

A from-scratch implementation of a modern large language model in PyTorch.

## what's here

- [x] Transformer decoder (GPT-style autoregressive language model)
- [x] Multi-head self-attention
- [x] Multi-head latent attention (MLA)
- [x] Rotary positional embeddings (RoPE)
- [x] RMSNorm
- [x] SwiGLU feed-forward network
- [x] Mixture of Experts (MoE) with top-k routing
- [x] Byte-pair encoding (BPE) tokenizer
- [x] AdamW optimizer with weight decay
- [x] Training loop with gradient clipping and cosine LR schedule
- [x] Cross-entropy loss with vocabulary projection
